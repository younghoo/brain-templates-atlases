# [Desikan-Killany](http://surfer.nmr.mgh.harvard.edu/fswiki/CorticalParcellation)

## 构建方法

根据沟回信息手工分割40个被试的皮层数据得到的概率地图，包含68个皮层脑区（每个半球各34个）。
