# Bellec2010

## 构建方法

* 根据大约200个被试静息态fMRI数据，对个体被试时间序列的相似性进行聚类，再采用bootstrap的方法分析聚类结果的稳定性，通过对稳定性进行聚类将全脑划分为7/12/20/36/64/122/197/325/444个脑区。
