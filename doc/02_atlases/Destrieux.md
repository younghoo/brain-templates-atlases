# Destrieux

## 构建方法

* 根据沟回信息手工分割12个被试的皮层数据得到的概率地图，包含148个皮层脑区（每个半球各74个）。

## 下载地址

* 内置于[FreeSurfer](http://surfer.nmr.mgh.harvard.edu/fswiki/FreeSurferWiki)软件包内。
