# DKT

## 构建方法

* 根据沟回信息手工分割40个被试的皮层数据，包含62个皮层脑区（每个半球各31个）。

## 下载地址

* <https://mindboggle.info/data.html>
* 内置于[FreeSurfer](http://surfer.nmr.mgh.harvard.edu/fswiki/FreeSurferWiki)软件包内。
