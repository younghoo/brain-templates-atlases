# Gordon2016

## 构建方法

* 基于120个被试的静息态fMRI数据，先计算每个被试的功能连接相似性矩阵，并计算相似性矩阵的梯度，然后对所有被试的梯度进行平均， 最后根据平均梯度将大脑皮层划分成333个脑区。
